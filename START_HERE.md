# 🚀 START HERE - Quick Launch Guide

## Welcome! Let's get your love story website running in 3 simple steps:

---

## Step 1: Install Everything ⚙️

Open your terminal in this folder and copy-paste this command:

```bash
npm install
```

⏱️ This will take 2-3 minutes. Go grab a coffee! ☕

---

## Step 2: Add Similoluwa's Photos 📸

You have 4 beautiful photos to add:

1. Go to the `public/gallery/` folder
2. Save the photos as:
   - `simi1.jpg` - Black outfit photo
   - `simi2.jpg` - Yellow top photo (first)
   - `simi3.jpg` - Yellow top photo (second)
   - `simi4.jpg` - White hoodie photo

💡 **Tip**: Just drag and drop the photos into the `public/gallery/` folder and rename them!

---

## Step 3: Launch! 🎉

Run this command:

```bash
npm run dev
```

Then open your browser and go to:

### 👉 **http://localhost:3000**

## 🆘 Having Issues?

### Website won't start?
Make sure you have Node.js installed: https://nodejs.org/
(Download version 18 or higher)

### Photos not showing?
- Check they're in the `public/gallery/` folder
- Make sure file names are exactly: `simi1.jpg`, `simi2.jpg`, etc.
- File names are case-sensitive!
