## 🎨 Design Features

- **Dark Luxury Theme**: Deep black (#0a0a0f) with midnight blue
- **Glassmorphism**: Frosted glass effects throughout
- **Neon Glow**: Purple (#9d4edd) and pink (#ff006e) accents

  - Playfair Display (serif) for romantic text

```bash

```

### 2. Add Photos
Save Similoluwa's 4 photos in `public/gallery/` as:
- `simi1.jpg` (black outfit)

