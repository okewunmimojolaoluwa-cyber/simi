## 🚨 Common Issues & Fixes

### Photos not showing?
```bash
# Make sure files are named exactly:
public/gallery/simi1.jpg
public/gallery/simi2.jpg
public/gallery/simi3.jpg
public/gallery/simi4.jpg
```

### Website won't start?
```bash
# Try reinstalling:
rm -rf node_modules
npm install
npm run dev
```

### Animations laggy?
- Close other browser tabs
- Use Chrome for best performance
- Check if computer is running other heavy apps

### Admin login not working?
- Password is case-sensitive: `emmjay`
- Tap logo exactly 5 times within 2 seconds
- Try refreshing the page
